📌 What Are Control Statements?

Control statements are used to control the flow of execution in a C program.

Normally, a program executes from top to bottom.
Control statements change this flow based on conditions or repetitions.

🔄 What Are Iteration Control Statements?

Iteration Control Statements are statements that execute a block of code repeatedly until a condition becomes false.

They are commonly called loops.

📂 Types of Iteration Statements in C

for loop

while loop

do-while loop

In this repository, we are learning about the for loop.

🔹 What Is a For Loop?

A for loop is an entry-controlled iteration statement used when the number of repetitions is known in advance.

It combines initialization, condition checking, and increment/decrement in a single line.

❓ Why Do We Use For Loop?

We use the for loop:

To reduce code repetition

When the number of iterations is known

To write clean and structured looping code

To perform counter-based repetition

📍 Where Do We Use For Loop?

For loops are used in:

Printing number series

Pattern programs

Iterating arrays

Performing repeated calculations

Menu-driven programs

Automation tasks

🧠 Syntax of For Loop
for(initialization; condition; increment/decrement)
{
    // block of code
}
💻 Example Program
#include <stdio.h>

int main()
{
    for(int i = 1; i <= 5; i++)
    {
        printf("%d\n", i);
    }

  return 0;
}
Output:
1
2
3
4
5
